# Decodelabs_Rule-Based-AI-ChatbotRule-Based AI Chatbot 🤖

Project Overview

This project is a simple Rule-Based AI Chatbot developed using Python. The chatbot responds to predefined user inputs using if-else statements and runs continuously until the user chooses to exit.

Objective

The main objective of this project is to understand the basic concepts of Artificial Intelligence, decision-making logic, control flow, and user interaction through a chatbot.

Features

- Responds to greetings such as "hello" and "hi"
- Answers simple predefined questions
- Handles exit commands like "bye"
- Runs in a continuous loop
- Displays a default response for unknown inputs

Technologies Used

- Python
- If-Else Statements
- While Loop

Working

1. The user enters a message.
2. The chatbot checks the input using if-else conditions.
3. If the input matches a predefined rule, the chatbot returns the corresponding response.
4. If the input does not match any rule, the chatbot displays a default message.
5. The chatbot continues running until the user enters "bye".

Sample Output

You: hello

Bot: Hi! How can I help you?

You: how are you

Bot: I am fine. Thank you!

You: bye

Bot: Goodbye!

Learning Outcomes

- Understanding of rule-based systems
- Implementation of decision-making logic
- Working with loops and conditional statements
- Introduction to basic Artificial Intelligence concepts

Conclusion

The Rule-Based AI Chatbot is a beginner-friendly AI project that demonstrates how machines can simulate simple conversations using predefined rules. This project helps build a strong foundation for more advanced AI and Machine Learning applications.
